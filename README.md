# PORTFOLIO-Project
A personal portfolio website built using pure HTML only.

This project showcases my:

* About section
* Technical skills
* AI/ML projects
* Education details
* Achievements
* Contact information

The portfolio is designed without using CSS or JavaScript to strengthen core HTML concepts and webpage structuring skills.

## 🚀 Features

* Clean and structured HTML layout
* Navigation bar with internal linking
* Skills table
* Project showcase section
* Contact form
* Responsive meta viewport support
* Semantic HTML tags

## 💻 Technologies Used

* HTML5

## 📂 Sections Included

* Header
* About Me
* Skills
* Projects
* Education
* Achievements
* Contact
* Footer

## 🎯 Purpose of This Project

This project was created to practice:

* HTML fundamentals
* Semantic tags
* Tables and forms
* Internal page navigation
* Structuring a portfolio website
